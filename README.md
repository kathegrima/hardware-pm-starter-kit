## ğŸ­ The Hardware Development Lifecycle

**Standard Phase Flow**: Concept > EVT > DVT > PVT > Mass Production > Maintenance > End-of-Life

| Phase | Duration | Purpose | Key Deliverables |
|-------|----------|---------|------------------|
| **Concept** | 4-8 weeks | Validate feasibility | PRD, initial BOM |
| **EVT** | 8-16 weeks | Prove functionality | Functional prototype, testing |
| **DVT** | 12-20 weeks | Validate design | Production-intent design, compliance |
| **PVT** | 8-16 weeks | Validate manufacturing | Pilot run, process validation |
| **MP** | Ongoing | Volume production | Shipping products, field data |

**Learn more**: [Phase Gate Deep Dive Guide](guides/03-phase-gate-guide.md)

## ğŸ’¡ Key Concepts

### Hardware vs Software PM

| Aspect | Software | Hardware |
|--------|----------|----------|
| **Iteration** | Days-weeks | Months-quarters |
| **Changes** | Can patch | Cannot modify post-mfg |
| **Cost** | Near zero updates | High tooling/inventory costs |
| **Sprints** | 2-4 weeks | 4-8 weeks |

### Critical Milestones

- **Design Freeze**: No changes without formal ECO (typically before DVT/PVT)
- **Tooling Commitment**: $50K-$500K investment (after DVT validation)
- **Component Lead Times**: 8-20 weeks for ICs, plan accordingly
